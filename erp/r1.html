Bem-vindo ao YLog

O YLog é um sistema de gerenciamento de logística para armazenamento e fretes. Aqui você pode realizar diversas tarefas relacionadas a pedidos, cadastro de usuários, veículos e outras rotinas.

Menu de Navegação

  • Início - Página atual que exibe uma mensagem de boas-vindas e explicações sobre cada menu.
  • Pedidos - Acesse essa rotina para adicionar ou consultar pedidos.
  • Cadastrar Usuário - Utilize essa rotina para cadastrar novos usuários com permissões específicas.
  • Cadastrar Veículos - Nessa rotina, é possível cadastrar os veículos utilizados no transporte.
